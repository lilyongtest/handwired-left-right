## Features
- Hybrid layout design: Row-staggered left half paired with ortholinear right half for optimal right-handed ergonomics
- Center-mounted OLED display for real-time status and information visualization
- Rotary encoder for convenient control of volume, navigation, or custom functions
- Ergonomic asymmetrical design that leverages the strengths of both staggered and ortholinear layouts
- Split keyboard configuration with left and right halves for comfortable typing posture
- Customizable firmware with full programmability for personalized key mapping and functions
